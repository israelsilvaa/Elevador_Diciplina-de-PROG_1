Como executar o projeto de forma correta, tanto no computador (Win10) quanto no Android.

Android: Aplicativo recomendado para execução "Pydroid 3" , disponível na Playstore https://play.google.com/store/apps/details?id=ru.iiec.pydroid3.
Dentro do app Pydroid 3, selecione o arquivo Python que deseja carregar, antes de executar certifique-se de ativar e utilizar o modo paisagem do seu 
dispositivo e aplicativo, pois do contrário a saída do projeto pode não ser correta.

Computador (Windows 10): compatível com qualquer IDE(PyCharm, Netbeans, Eclipse) em versão do python mais recente possível.
