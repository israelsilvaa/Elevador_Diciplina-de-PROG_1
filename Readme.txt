Como executar o projeto de forma correta, tanto no computador(Win10) quanto no Android

Andoid: Aplicativo recomendado para execução "Pydroid 3" , disponivel na Playstore <https://play.google.com/store/apps/details?id=ru.iiec.pydroid3>

dendo do app Pydroid 3, selecione o arquivo Python que deseja caregar, antes de executar certifique-se de ativar e ulizizar o modo paisagem do seu
dispositivo e aplicativo, pois do contrario a saida do projeto pode não ser correta.

Computador(Windowns 10): compativel com qualquer IDE(PyCharm, Netbeans, Eclipse) em versão do python mais recente possivel.
